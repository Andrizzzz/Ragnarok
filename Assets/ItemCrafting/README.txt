Inventory and Item Crafting System in Unity

This project contains an example of a simple Inventory and Item Crafting System in Unity.

- Open 'TestScene' from the Scenes folder to launch an example scene
- Click on the element in the inventory then drop it in the crafting table by clicking on the specific slot(s), return the selected item by clicking back on the inventory
- Check the video from the tutorial below for the predefined crafting recipes, and how to add custom recipes
